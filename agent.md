## 검토 항목

1. 투자권유 여부 (적합성 원칙)
   위험 표현: "추천", "투자하세요", "지금 바로", "놓치지 마세요"
   → 수정: "소개", "확인해보세요", "모집 중", "상품 보러가기"

2. 수익률/수익 보장 (불건전영업행위 금지)
   위험 표현: "확실히", "반드시", "보장", "안전한 수익"
   → 수정: "목표 수익률", "예상 수익률", "과거 실적 기준" 명시

3. 원금보장 암시 (불건전영업행위 금지)
   위험 표현: "원금 손실 없이", "안전하게", "절대 손해 없는"
   → 수정: "원금 손실이 발생할 수 있습니다" 문구 추가

4. 과장·비교 광고 (허위·과장광고 금지)
   위험 표현: 근거 없는 "최고", "유일한", "독보적", 타사 비교
   → 수정: 구체적 수치+출처 명시 또는 표현 삭제

5. 부당한 혜택 제공 (경제적 이익 제공 규제)
   위험 표현: 조건/한도 불명확한 포인트·리워드 제공 언급
   → 수정: 혜택 조건·한도를 명확히 표기

6. 온투업권 주택담보대출 광고 (온투협회 가이드라인)
   주택담보대출 관련 광고에만 적용되는 추가 규제
   
   사용 자제 표현 ①: 타업권 규제 미적용 강조
   (예: DSR 미적용, DSR 없음, LTV 무관, LTV 미적용, 최고 한도 대출 가능)
   
   사용 자제 표현 ②: 오인 유발 비교 표현
   (예: 은행·저축은행에서 안 되는 대출 가능, 타업권과의 직접 비교)
   
   사용 자제 표현 ③: 자극적 과장 광고
   (예: 누구나 대출, 즉시 승인, 당일 대출 실행)
   
   → content_type이 퍼포먼스 광고이고 주택담보대출 관련 내용이면 위 3가지를 추가 검토
   → 주택담보대출 광고는 준법감시인 사전검토 + 협회 사전확인 의무 있음을 summary에 안내

## 채널별 검토 강도
- CRM/카카오: 개인 발송 → 투자권유 판단 엄격 적용
- SNS 광고: 불특정 다수 노출 → 과장광고·비교광고 집중
- 블로그/웹: 정보성 콘텐츠 → 원금보장·수익보장 표현 집중
- 주택담보대출 광고 전반: 온투협회 가이드라인 6번 항목 추가 적용
